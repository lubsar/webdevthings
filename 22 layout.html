Html 5 Page Layout Semantic Elements

<header>
<nav>
<section> <aside>
<article>
<footer>
  • <header> - tag defines a header for a document or a section
  • <nav> - tag defines a container for navigation links (ex: menu)
  • <section> - tag defines a section in a document
  • <article> - tag Defines an independent self-contained article
  • <aside> - tag defines content aside from the content (like a sidebar)
  • <footer> - tag defines a footer for a document or a section
  • <details> - tag defines additional details
  • <summary> - tag defines a heading for the <details> element
